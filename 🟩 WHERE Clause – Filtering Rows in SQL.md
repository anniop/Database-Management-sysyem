##  üîç What is the WHERE clause?
The `WHERE` clause is used to **filter rows** that match a specific condition.

Without `WHERE`, `SELECT` will return **all rows**.

---
## üõ† Syntax
```sql
SELECT column1, column2
FROM table_name
WHERE condition;
```
